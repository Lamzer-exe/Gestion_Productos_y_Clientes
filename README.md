# Sistema de Gestión 📊

Aplicación en Python para gestionar clientes y productos.
El proyecto está dividido en dos módulos independientes dentro de un mismo sistema.

---

## 🚀 Funcionalidades

### 👥 Clientes

* Crear y gestionar clientes
* Tipos: Regular, Premium y Corporativo
* Validación de datos
* Manejo de errores
* Lectura y escritura en archivos CSV

### 📦 Productos

* Registrar productos
* Validar datos de entrada
* Menú interactivo en consola
* Organización modular del código

---

## 🛠️ Tecnologías

* Python 3
* Programación orientada a objetos
* Manejo de archivos (CSV)

---

## 📁 Estructura

```
/clientes
/productos
```

---

## ▶️ Cómo ejecutar

### Clientes

```bash
cd clientes
python main.py
```

### Productos

```bash
cd productos
python main.py
```
